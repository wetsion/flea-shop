## Flea Shop

跳蚤商店

#### 背景

针对公司跳蚤群中的出售物品、出饭，以及在节假日时，出售公司节日礼盒的情况，发现缺少一个统一的平台。

大致可讲物品分为两类：秒杀类和竞拍类。

#### 目标功能

 - 核心功能
 
    - 物品的上架，包含设置价格、数量、有效时间
 
    - 物品的浏览，用户可以在列表页浏览以及搜索物品，包含有效和过期的物品
    
    - 物品的竞购，选择物品，进行竞购或者参与秒杀
    
 - 辅助功能
 
    - 用户信息
    
 - 后期可选功能
 
    - 登录，接入微信或企业微信OA扫码登录
 
    - 订阅，用户可按关键字或种类订阅物品上架信息
    
    - 通知，竞购结果以邮件或者企业微信方式通知
    
    - 支付，接入微信支付或支付宝在成功之后直接支付
